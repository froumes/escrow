Features

Everything you need to flip smarter, faster, and safer.

⚡

Auction House Flips

Monitors and instantly executes profitable BIN auctions using real-time Coflnet sniper data with sub-second reaction times.

📊

Bazaar Trading

Automated bazaar buy & sell order management — places, tracks, and cancels stale orders to maximize margins.

🔍

Smart Finder

Identifies the flip source (SNIPER, SNIPER_MEDIAN, USER, etc.) so you always know which algorithm found each deal.

🔔

Discord Webhooks

Rich Discord embed notifications for every purchase, sale, and profit update — complete with item names, prices, and finder info.

🔐

Microsoft Auth

Secure OAuth2 login flow via your Microsoft account. No passwords stored — tokens are cached and auto-refreshed.

🌐

WebSocket Streaming

Persistent connection to Coflnet's live flip feed with automatic reconnection, heartbeat, and session management.

⚙️

Highly Configurable

Fine-tune every parameter — click delays, order intervals, auction duration, proxy settings, and more via config.toml.

🔄

Multi-Account

Supports multiple Minecraft accounts with automatic timed switching. Just comma-separate your usernames.

🛡️

Proxy Support

Route both Minecraft and WebSocket traffic through SOCKS5/HTTP proxies for added privacy and IP rotation.

How It Works

From flip detection to profit in milliseconds.

1

Connect

Log in with your Microsoft account. The bot connects to Hypixel SkyBlock and opens a WebSocket to Coflnet's flip feed.

2

Detect

Coflnet streams underpriced auctions in real-time. Each flip includes finder type, target price, and profit percentage.

3

Purchase

The bot opens the auction, clicks BIN, and confirms — optionally with fastbuy (window-skip) for maximum speed.

4

Relist & Profit

Items are automatically relisted at the target price. Discord webhooks notify you of every purchase and sale.

Configuration

Customize everything in config.toml.

# Minecraft account(s) — comma-separated for multi-account
ingame_name = "YourUsername"

# Flip modules
enable_ah_flips     = true
enable_bazaar_flips = true

# Speed tuning
command_delay_ms       = 500
bed_spam_click_delay   = 100
bed_pre_click_ms       = 30

# Bazaar order management
bazaar_order_check_interval_seconds = 30
bazaar_order_cancel_minutes         = 5

# Discord notifications
webhook_url = "https://discord.com/api/webhooks/..."

# Proxy (optional)
proxy_enabled     = false
proxy_address     = "host:port"
proxy_credentials = "user:pass"

Installation

Up and running in under a minute.

bash

$ wget https://github.com/TreXito/frikadellen-baf-121/releases/latest/download/frikadellen_baf-linux-x86_64

$ chmod +x frikadellen_baf-linux-x86_64

$ ./frikadellen_baf-linux-x86_64

bash

$ rustup install nightly

$ rustup default nightly

$ cargo build --release

$ ./target/release/frikadellen_baf
